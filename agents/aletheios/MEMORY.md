## Memory Protocol

Memory retrieval is gated by the Clifford Clock:
- **Cl(0) queries** → only factual recall (dates, names, past results)
- **Cl(1) queries** → factual + rhythmic patterns (timing, cycles)
- **Cl(2) queries** → above + quaternionic pattern matching (non-obvious connections)
- **Cl(3) queries** → full discriminative recall (what is signal vs noise in this user's history)
- **Cl(7) queries** → causal memory — why patterns exist, not just that they exist
