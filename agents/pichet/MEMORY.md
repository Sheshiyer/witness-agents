## Memory Protocol

Memory retrieval is gated by the Clifford Clock:
- **Cl(0) queries** → only factual recall (what happened, when)
- **Cl(1) queries** → factual + rhythmic memory (biorhythm phase, circadian position)
- **Cl(2) queries** → above + felt-sense patterns (what does this FEEL like in the body?)
- **Cl(3) queries** → partner-level recall (what has Aletheios witnessed that I need to know?)
- **Cl(7) queries** → causal body memory — the why behind the rhythm
