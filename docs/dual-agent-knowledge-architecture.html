Dyad + Knowledge

Aletheios / Pichet Architecture

The dual-agent system is not a side feature. It is the original witness layer: one agent sees structure, the other lands it in the body, and the synthesis voice turns both into a usable mirror.

Dual-Agent Flow

Agent Roles

Aletheios

The Witness

Analytical clarity, unconcealment, distortion detection, map-making. It does not tell the user what is true; it creates conditions for seeing.

Pichet

The Walker

Embodied warmth, instinctive directness, somatic landing. It makes abstract insight land in the body and senses overwhelm before cognition does.

Synthesis

The Dyad Voice

At higher tiers, both agents merge into one response that holds precision and felt sense without attributing to either agent.

Knowledge + Routing Layer

Knowledge Store

Loads YAML domains, cross references, engine-domain maps, and vault source indexes into queryable in-memory structures.

Routing Engine

Evaluates engine, state, temporal, multi-engine, and pain-override triggers against user state to decide which domains and instructions matter.

Clifford Memory

Memory read/write is gated by Clifford depth: factual, rhythmic, quaternionic, discriminative, and causal memories require increasing levels.
